## 🌸 Features

- **Secret Passcode Screen** — Enter `0107` to unlock 💕
- **Birthday Celebration Page** — Rotating text, dancing Dudu bear with balloon & cake
- **Birthday Diary** — Typewriter love letter with realistic page-turn effect
- **Floating bears, hearts, confetti** throughout all pages
- **Heart cursor effect** — Custom cursor made of hearts
- **Background music** — Soft romantic melody toggle
- **Mobile responsive** — Works beautifully on all devices
- **GitHub Pages ready** — Pure HTML/CSS/JS, no build step needed

## 🎯 How to Deploy to GitHub Pages

1. Create a new GitHub repository (e.g., `bubu-birthday`)
2. Upload all 3 files: `index.html`, `style.css`, `script.js`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch, `/ (root)` folder
5. Click **Save** — your site will be live at `https://yourusername.github.io/bubu-birthday`
